# Design Notes Archive

This folder keeps secondary design notes, deprecated split documents, and historical planning material that has been consolidated into the main documentation set.

The root `docs/` folder should stay focused on the long-term maintained documents:

- `PRODUCT.md`
- `ARCHITECTURE.md`
- `MIGRATION_SYSTEM.md`
- `CATALOG_SYSTEM.md`
- `CONFIG_AND_SECURITY.md`
- `ROADMAP.md`
- `DEPLOY.md`
- `DEPLOY_SELF.md`

Files in this archive should either point to the consolidated document that replaced them or preserve background material that is still useful but not part of the primary documentation path.
